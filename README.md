Descripciòn breve del proyecto
Este proyecto consiste en una página web desarrollada para presentar información de restaurantes. Incluye varias secciones. 

Estructura de archivos 
GastroFinder es una aplicación web sencilla que permite explorar, buscar y registrar restaurantes de la ciudad de Medellín.
El proyecto cuenta con tres páginas principales:

Inicio (index.html): muestra una lista de restaurantes destacados.

Buscar (search.html): permite filtrar restaurantes por nombre en tiempo real.

Nuevo (new.html): permite agregar nuevos restaurantes a la lista.

El objetivo es practicar estructura HTML, estilos CSS, navegación entre páginas, manejo de datos y documentación del proyecto.

GastroFinder/
│── index.html          # Página principal
│── search.html         # Página de búsqueda
│── new.html            # Página para agregar restaurantes
│── styles.css          # Estilos del proyecto
│── js/
│    └── main.js        # Lógica básica en JavaScript
│── images/             # Imágenes de los restaurantes (Parrilla.jpg, Mar.jpg, Vegetariano.jpg, Restauramte2.jpg)
│── README.md           # Documentación del proyecto

Instrucciones para ejecutar el proyecto
Descarga o clona el proyecto.
Abre la carpeta del proyecto.
Haz doble clic en el archivo index.html o
click derecho, Abrir con tu navegador.
